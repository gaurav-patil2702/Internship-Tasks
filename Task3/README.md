
# AI Chatbot with NLP

## Objective
Build a chatbot using Natural Language Processing libraries like NLTK.

## Tools Used
- Python
- NLTK

## Features
- Tokenization
- Stopword removal
- Basic NLP preprocessing
- Rule-based response system

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the chatbot:
   python chatbot.py

3. Type messages and interact with the bot.
   Type 'bye' to exit.

## Deliverables
- chatbot.py (Main Script)
- requirements.txt
- README.md
