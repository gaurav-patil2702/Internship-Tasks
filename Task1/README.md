
# API Integration and Data Visualization

## Objective
Fetch data from a public API and visualize it using Python.

## Tools Used
- Python
- OpenWeatherMap API
- Matplotlib

## Steps to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Add your OpenWeatherMap API key in weather_dashboard.py
3. Run the script:
   python weather_dashboard.py

## Output
A line chart showing temperature forecast for the next 24 hours.
